Takes a Porter Stemmer and a directory full of .json files to populate an inverted index that is allocated on the heap. This method feeds the text file into a stringstream and passes the stringstream into a boost, json property tree. The tokens are individually transformed to lowercase and stemmed before being put into the inverted index.
